Detailed Description

Boundary strategy base class.

Defines the interface for boundary-type classes.

BoundaryStrategy classes implement methods specific to the boundary type of the model. The simulation area is assumed to have boundary conditions at x = side, y = side, and have ranges [0, side] in all directions.
