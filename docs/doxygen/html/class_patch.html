void Patch::adults_die ( )

Removes dying adults from the patch.

Determines the number of adults that die in the given day and removes them from the patch. The number of males that die (for a given genotype) is determined by a binomial distribution of adult mortality, and similarly for virgin and mated females.

void Patch::juv_eclose ( )

Turns the oldest juveniles into adults.

Note
Not all juveniles survive eclosion.

The number of survivors is determined by a binomial distribution of the juvenile survival probability. Sex determination upon eclosion is determined by a binomial distribution with 0.5 probability.

Females lay eggs, creating new juveniles.

◆ virgins_mate()

void Patch::virgins_mate ( )

Mates a fraction of the virgin females in the patch.

Determines the number of virgin females that mate in the given day (for a given female genotype) with a male of genotype j by using a binomial distribution of the mating rate. Then, tranforms the virgin females into mated females carrying male sperm of genotype j. Mating is carried out within the patch, and females only mate once.

See also
Patch::mate_rate
