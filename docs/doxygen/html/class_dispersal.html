Detailed Description

Dispersal base class.

Defines the interface for dispersal classes.

The Dispersal classes compute the connections between Patch objects with their respective connection weights, dependent on the maximum dispersal distance max_dist, and carry out dispersal in and out of the patches, dependent on the dispersal rate disp_rate.

Determines the number of mated females (of each genotype combination) dispersing out from each patch.

The number of females of a given genotype combination dispersing from a given patch is determined by a random draw from a binomial distribution with probability of adult dispersal rate.
