\doxysubsection{Detailed Description}
Records model data. 

Creates Coordinate\+List, Totals and Local\+Data output .txt files for each repetition of the model. Coordinate\+List\+: contains the coordinates of all recorded sites. Totals\+: contains the total numbers of males of each genotype across all sites, for each recorded day of the simulation. Local\+Data\+: contains the number of males of each genotype for each site on each simulated day that is recorded. \begin{DoxySeeAlso}{See also}
