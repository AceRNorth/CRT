\doxysubsection{Detailed Description}
Boundary strategy base class. 

Defines the interface for boundary-\/type classes.

\doxylink{class_boundary_strategy}{Boundary\+Strategy} classes implement methods specific to the boundary type of the model. The simulation area is assumed to have boundary conditions at x = {\ttfamily side\+\_\+x}, y = {\ttfamily side\+\_\+y}, which are calculated from provided input coordinates or assumed to be 1 otherwise.
